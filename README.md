🚀 NEXU Toolkit

📖 Overview
NEXU is an Android/iOS hacking toolkit that provides a set of tools to test mobile device security and learn how cybersecurity works.

✨ Features

Brute-force PIN (4 and 6 digits)

Lock screen bypass using wordlists or antigard bypass

Android root via SuperSU

ADB tools

Data reset

Lock screen removal (Root required)

Metasploit backdoor

Control Android via Scrcpy

Phone information

IP logger and webcam capture over the internet

Firestore vulnerability scan


📌 Usage
After installation, run the program with:

python3 nexu.py

Then choose the appropriate number from the menu to use the desired feature.
